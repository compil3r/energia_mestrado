Como Jogar

🎯 Objetivo do Jogo

Este jogo foi desenvolvido para ensinar conceitos fundamentais de lógica de programação de forma divertida e interativa. Você é um técnico responsável por restaurar a energia da cidade através da resolução de problemas lógicos.

⚡ Fase 1: Algoritmos

Conceito: Aprenda sobre sequências lógicas e algoritmos básicos.

Objetivo: Conecte os cabos seguindo a sequência correta para restaurar a energia das estações.

Como jogar:

  • Clique nos cabos na ordem correta
  • Siga a lógica da sequência apresentada
  • Complete todas as conexões para avançar

🔀 Fase 2: Condicionais

Conceito: Aprenda sobre estruturas condicionais (if/else) e tomada de decisões.

Objetivo: Analise as condições dos painéis e tome as decisões corretas para restaurar a energia.

Como jogar:

  • Observe os indicadores dos painéis
  • Tome decisões baseadas nas condições apresentadas
  • Use lógica condicional para resolver os problemas
